Sobre o Sistema

O Race Live demonstra comunicação em tempo real utilizando WebSocket e arquitetura MVC para sincronizar múltiplos clientes simultaneamente.

📡 WebSocket

Atualização instantânea do placar para todos os usuários conectados.

🏁 Tempo Real

Ranking recalculado automaticamente a cada nova volta registrada.

🧱 Arquitetura MVC

Separação entre Models, Controllers, Rotas e Views.

💾 Persistência

Corridas armazenadas em JSON com sincronização automática.

⚡ Baixa Latência

Broadcast imediato usando WebSocket Server.

📱 Responsivo

Interface adaptável para desktop e dispositivos móveis.

Funcionalidades

🏁 Criar Corrida

🏎️ Registrar Voltas

🏆 Ranking Automático

Classificação recalculada automaticamente.

📡 Sincronização Live

📊 Histórico

Consulta completa das corridas anteriores.

🔌 API REST

Dados acessíveis via endpoints JSON.

Tecnologias Utilizadas

Node.js Express WebSocket MVC REST API HTML5 CSS3 JSON
